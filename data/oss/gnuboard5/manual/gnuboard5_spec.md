## 1. 회원 관리

### 1.1 회원가입
- 아이디, 비밀번호, 이름, 이메일, 닉네임 입력 필수
- 아이디: 영문/숫자 조합, 중복 불가, 금지단어 사용 불가
- 비밀번호: 최소 6자 이상
- 이메일 형식 검증 필수
- 약관 및 개인정보처리방침 동의 필수
- 이메일 인증 옵션 (관리자 설정에 따라 활성화)
- 레벨 1로 가입, 관리자가 레벨 조정 가능

### 1.2 로그인 / 로그아웃
- 아이디 + 비밀번호로 로그인
- 로그인 실패 시 오류 메시지 표시
- 비로그인 상태에서 회원 전용 기능 접근 시 로그인 페이지로 이동
- 로그인 유지 (쿠키) 옵션 지원
- 로그아웃 후 메인 페이지로 이동

### 1.3 정보 수정
- 비밀번호 확인 후 개인정보 수정 가능
- 수정 가능 항목: 비밀번호, 이름, 이메일, 닉네임, 홈페이지, 서명, 자기소개
- 회원 탈퇴 기능 제공

### 1.4 비밀번호 찾기
- 가입 이메일로 임시 비밀번호 발송
- 이메일 미입력 또는 불일치 시 오류 표시

## 2. 게시판

### 2.1 게시글 목록 조회
- 게시판별 글 목록 표시 (페이지네이션)
- 기본 정렬: 최신 글 순서 (등록일시 내림차순)
- 목록에 표시: 번호, 제목, 작성자, 날짜, 조회수, 댓글 수
- 공지글은 목록 상단 고정 표시
- 검색 기능: 제목, 내용, 작성자 검색 가능
- 페이지당 글 수 관리자 설정 가능

### 2.2 게시글 작성
- 비로그인 시 작성 불가 (설정에 따라 비회원 작성 허용 가능)
- 제목 필수, 내용 선택
- HTML 에디터 또는 텍스트 에디터 사용
- 파일 첨부 기능 (최대 개수·용량 관리자 설정)
- 비밀글 설정 (작성자 및 관리자만 열람)
- 공지글 설정 (관리자 권한)
- 작성 후 포인트 지급 (관리자 설정)

### 2.3 게시글 상세 조회
- 제목, 내용, 작성자, 작성일, 조회수 표시
- 조회수 중복 방지 (동일 IP 동일 글 카운트 제한)
- 첨부파일 목록 및 다운로드 링크 표시
- 비밀글: 작성자·관리자 외 내용 비공개
- 이전/다음 글 이동 링크

### 2.4 게시글 수정
- 작성자 본인 또는 관리자만 수정 가능
- 비밀번호로 작성된 글은 비밀번호 확인 후 수정
- 수정 이력 없음 (버전 관리 미지원)

### 2.5 게시글 삭제
- 작성자 본인 또는 관리자만 삭제 가능
- 삭제 시 확인 다이얼로그 표시
- 삭제 후 목록으로 이동

### 2.6 댓글
- 게시글당 댓글 작성 가능
- 댓글 내용 필수
- 비밀 댓글 옵션
- 댓글 수정·삭제: 작성자 또는 관리자만 가능
- 대댓글(답글) 지원
- 댓글 작성 시 포인트 지급 (관리자 설정)

### 2.7 파일 첨부 및 다운로드
- 허용 확장자 관리자 설정
- 다운로드 시 포인트 차감 (관리자 설정)
- 비로그인 사용자 다운로드 제한 (관리자 설정)

## 3. 검색

### 3.1 통합 검색
- 제목, 내용, 작성자 검색 가능
- 게시판별 또는 전체 검색
- 검색 결과 페이지네이션

## 4. 포인트 시스템

### 4.1 포인트 적립 및 사용
- 회원가입, 로그인, 게시글 작성, 댓글 작성 시 포인트 적립
- 파일 다운로드 시 포인트 차감
- 포인트 내역 조회 (회원 마이페이지)
- 포인트 부족 시 제한 기능 활성화 (관리자 설정)
- 포인트 사용/적립 규칙은 관리자 설정

## 5. 관리자 기능

### 5.1 기본 환경 설정
- 사이트 제목, 관리자 이메일 설정
- 포인트 시스템 활성화/비활성화
- 에디터 선택 (smarteditor, ckeditor 등)
- 회원가입 설정 (약관, 이메일 인증)
- 접속 IP 차단 관리

### 5.2 회원 관리
- 전체 회원 목록 조회 및 검색
- 회원 정보 수정 (레벨, 이메일, 포인트 등)
- 회원 강제 탈퇴
- 회원 레벨별 권한 설정 (1~10)
- 접근 금지 설정

### 5.3 게시판 관리
- 게시판 그룹 생성·삭제
- 게시판 생성·수정·삭제
- 게시판별 권한 설정 (읽기·쓰기·댓글·다운로드)
- 게시판별 포인트 설정
- 페이지당 글 수 설정

### 5.4 게시글 관리
- 특정 게시글 강제 삭제
- 공지글 설정·해제
- 게시글 이동 (다른 게시판으로)
- 게시글 복사

### 5.5 메뉴 관리
- 메인 메뉴 항목 추가·수정·삭제
- 메뉴 순서 변경

---

## 6. 접근 제어 및 보안

### 6.1 레벨 기반 권한
- 회원 레벨 1~10 (10이 최고)
- 게시판별 최소 레벨 설정
- 비회원(레벨 0) 접근 범위 설정 가능

### 6.2 비밀글
- 게시글 및 댓글 비밀 설정
- 작성자·관리자 외 내용 비공개

### 6.3 IP 차단
- 관리자가 특정 IP를 접근 금지 목록에 추가
- 차단된 IP에서 접속 시 접근 불가 메시지 표시

---

## 7. 쇼핑몰 (영카트)

### 7.1 상품 관리
- 상품 등록·수정·삭제 (관리자)
- 상품 분류별 진열
- 재고 관리

### 7.2 주문 및 결제
- 장바구니 추가·수정·삭제
- 주문서 작성 (배송지, 결제 방법 선택)
- 결제 수단: 가상계좌, 신용카드 (PG 연동 필요)
- 주문 상태 조회 (회원)

---

## 8. 경계 조건 및 예외 상황

### 8.1 입력 길이 제한
- 아이디: 3~20자
- 비밀번호: 6~20자
- 닉네임: 2~20자 (바이트)
- 게시글 제목: 255자 이내
- 이메일: 표준 형식, 255자 이내

### 8.2 파일 업로드 제한
- 허용 확장자 외 파일 거부
- 파일 용량 초과 시 오류

### 8.3 중복 처리
- 중복 아이디 가입 시 오류
- 중복 이메일 가입 시 경고 (설정에 따라)
- 동일 IP 단시간 다중 게시글 작성 제한 (도배 방지)

# data/ — 운영·실증 데이터

> **상태:** PoC 시작 단계 (2026-05-19).

이 폴더는 AWT의 *실제 운영 흐름에서 발생하는 데이터*를 보관한다. 설계 단계의 `doc/`와 달리, 시간이 지나면서 누적되는 *살아있는 데이터*.

## 하위 폴더

| 폴더 | 내용 | 보관 정책 |
|---|---|---|
| `poc/` | PoC 시험 결과·기록 | 영구 (의사결정 audit) |
| `runs/` | 제품 시험 1회당 동결된 산출물 (`<run-id>/`) | 인증 기간 + 1년 |
| `metrics/` | 메트릭 누적 집계 | 영구 |
| `anonymized/` | RAG용 익명화 데이터 (Phase 2) | 영구 |
| `prompts/` | 호출된 prompt 전문·모델 버전 메타 | 영구 (audit) |

## 민감도

- 원본 고객사 데이터는 **이 폴더에 두지 말 것** (별도 안전 보관 영역)
- 자격 증명·PII는 *익명화 파이프라인 통과 후*에만 anonymized/ 진입
- L1+L2 익명화 자동, L3+L4 수동 검토 (D16)

# data/poc/ — PoC 결과 보관

각 PoC 세션마다 폴더 생성: `<YYYY-MM-DD>/`

폴더 내 권장 파일:
- `start-here.md` — 본 세션의 시작 가이드
- `result.md` — PoC 결과 기록 (`doc/03-architecture/06-poc-validation-plan.md` §8 양식)
- `prompts/` — 사용한 prompt 전문
- `screenshots/` — 결과 캡처
- `notes.md` — 자유 메모

PoC는 사용자가 직접 실행하므로 본 폴더의 파일은 *사용자가 작성*. AWT는 양식·가이드만 사전 제공.

# 샘플 게시판 사용자 매뉴얼 v1.0

## 1. 제품 개요

본 제품은 간단한 웹 기반 게시판이다. 로그인 후 글을 작성·조회·수정·삭제할 수 있으며, 다른 사용자의 글에 댓글을 남길 수 있다.

### 1.1. 시스템 요구사항
- 브라우저: Chrome 100 이상, Firefox 100 이상
- 자바스크립트 활성화 필요
- 화면 해상도: 1024 × 768 이상

## 2. 회원 인증

### 2.1. 로그인
"로그인" 버튼을 누르면 로그인 화면으로 이동한다. 아이디와 비밀번호를 입력하고 "로그인" 버튼을 누르면 로그인된다. 잘못된 정보 입력 시 오류 메시지가 표시된다.

### 2.2. 로그아웃
상단의 "로그아웃" 버튼을 누르면 로그아웃된다.

## 3. 게시글

### 3.1. 글 작성
로그인한 사용자만 글을 작성할 수 있다. 상단 "글쓰기" 버튼을 누르면 글쓰기 화면으로 이동한다. 제목과 내용을 입력하고 "등록" 버튼을 누르면 글이 등록된다.

- **제목은 적절한 길이로 작성해야 한다.**
- 내용은 비어 있을 수 없다.

### 3.2. 글 목록 조회
"목록" 버튼을 누르면 글 목록을 볼 수 있다. 한 페이지에 10개의 글이 표시되며, 페이지네이션으로 이동할 수 있다. 비로그인 사용자도 글 목록을 조회할 수 있다.

### 3.3. 글 상세 조회
목록에서 글 제목을 누르면 상세 화면으로 이동한다. 작성자, 작성일, 본문, 댓글이 표시된다.

### 3.4. 글 수정
**자신이 작성한 글만 수정할 수 있다.** 상세 화면에서 "수정" 버튼을 누르면 수정 폼으로 이동한다. 제목과 내용을 변경하고 "저장" 버튼을 누르면 수정된다.

### 3.5. 글 삭제
**자신이 작성한 글만 삭제할 수 있다.** 상세 화면에서 "삭제" 버튼을 누르면 확인 후 글이 삭제된다.

## 4. 댓글

### 4.1. 댓글 작성
로그인한 사용자만 댓글을 작성할 수 있다. 상세 화면에서 댓글 내용을 입력하고 "댓글 작성"을 누른다.

### 4.2. 댓글 삭제
상세 화면에서 각 댓글의 "삭제" 버튼을 눌러 댓글을 삭제할 수 있다.

## 5. 에러 처리

잘못된 입력이나 권한 오류 시 *적절한 한국어 안내*가 표시되어야 한다.

# tools/ — 실행 도구·스크립트

> **상태:** 비어 있음 — 구현 단계에서 채움

AWT의 자동 실행과 운영을 지원하는 도구들. Agent가 호출하거나 사람이 직접 실행한다.

## 예정 내용

- DOM/이미지 스캐너 실행기
- Playwright 기반 TC 실행기 (셀렉터 self-healing 포함)
- Mutation testing runner
- Traceability matrix 생성기
- 보고서 변환 도구 (TC JSON → 시험소 표준 양식)
- RAG 인덱싱/검색 CLI

설계 단계에서 도구의 인터페이스만 먼저 정의한다 (`doc/03-architecture/`).
